TruxModel

TruxModel

new TruxModel(name) → {Object}

A client side interface for a remote data model.

Each TruxModel is expected to have a unique id property.

Parameters:
Name Type Description
name String the name of this TruxModel
Source:
  • TruxModel.js, line 27
Returns:
this - this TruxModel
Type
Object
Examples
//basic usage
    var MyModel = new TruxModel('My Model');
//advanced usage
    var UserModel = function(data) {
        TruxModel.call(this);

        this.setData(data);

        this.getName = function () {
            return this.data.name;
        }

        this.setName = function (name) {
            this.data.name = name;
        }
    }
