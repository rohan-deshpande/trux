# Model properties

[`@see - Store properties`](/api/store/properties.md)

# `data`

```js
@prop {object|null}
```
The data which defines the model. Defaults to null.

# `collection`

```js 
@prop {boolean|object}
```

The collection the model belongs to. Defaults to false.
