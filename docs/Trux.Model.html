Model

Trux. Model

new Model(data) → {Object}

A client side interface for a remote data Model.

Each Model is expected to have a unique id property.

Parameters:
Name Type Description
data Object the data which defines this Model

Examples
//basic usage
       var MyModel = new Model('My Model');
//advanced usage
       var UserModel = function(data) {
           Trux.Model.call(this);

           this.setData(data);

           this.getName = function () {
               return this.data.name;
           }

           this.setName = function (name) {
               this.data.name = name;
           }
       }
